# Mani Group 研究対象まとめ

- **乱流とマクロ平均場モデルの精緻化**  
  - マクロ強制法 (Macroscopic Forcing Method; MFM) を用いて、平均場輸送を支配する非局所・異方的オペレーターを直接測定し、RANS や LES のモデル形の妥当性を検証・改良。  
  - チャネル乱流における運動量拡散の非局所・異方性を計測し、勾配拡散仮定を超える渦粘性テンソルを抽出（park_2024_jfm, liu_2024_prf）。  
  - Rayleigh–Taylor 不安定性に対するスカラー輸送の非局所拡散を測定し、高次モーメントを用いたモデル化を検討（lavacot_2024_jfm）。  
  - 異方的強制を用いた均一乱流データで応力崩壊モデルを再構築し、立方項まで拡張した減衰モデルを提案（homan_2024_prf）。  

- **多相流の数値モデリング**  
  - N 相の界面を扱う保存型フェーズフィールド法を開発し、質量・運動量・運動エネルギーの保存と多相接合部の正しい平衡形状を実現（mirjalili_2024_jcp）。  
  - 変動する界面厚さや表面張力を考慮した高精度スキームの設計により、多相流の離散的整合性を確保。  

- **電気動力学的流れ・電気化学システム**  
  - 電解・膜・プラズマ系での電圧駆動イオン輸送を、流体・質量輸送・電場の結合問題として理論・数値的に解明。  
  - 電気透析におけるカオス的電気対流を DNS で直接再現し、従来の簡略モデルとの差を定量化。現在は電気対流の物理と、膜・ガス拡散電極アセンブリでの触媒変換・輸送現象に注力。  

- **粒子混入流れと熱・放射輸送**  
  - タービンや太陽受熱器などで重要な粒子-乱流-放射の三者相互作用を調査し、粒子の濃度偏在や乱流・熱輸送の改変を定量化。  
  - 次世代の Euler–Lagrange 型粒子流れモデルを整備し、体積加熱や効率向上に資する設計指針を提供。  

- **設計指針**  
  - 高忠実度 DNS/LES と MFM・随伴法を組み合わせて、平均場オペレーターを直接同定し、モデルの「形」そのものを検証することで、乱流・多相・電気動力学的流れの統一的なクロージャを目指す。  
  - 得られた演算子の非局所性・異方性を明示化し、工学的に重要な壁乱流、分離流、環境流、混合・輸送プロセスへの応用を進める。
